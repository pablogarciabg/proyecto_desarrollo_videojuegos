# 🎮 MyGameStore - Proyecto Educativo

## 📚 Descripción del Proyecto

**MyGameStore** es un proyecto educativo diseñado para enseñar Clean Architecture en Android utilizando Jetpack Compose y las mejores prácticas recomendadas por Google.

**Concepto:** Una aplicación para explorar videojuegos (API externa), ver detalles y gestionar una biblioteca personal con estados como "Pendiente", "Jugando" y "Terminado".

**API Utilizada:** [RAWG.io](https://rawg.io/apidocs) - API gratuita con excelente documentación e imágenes de alta calidad.

---

## 🎯 Objetivos Pedagógicos

1. **Datos Remotos (Retrofit):** Consumir APIs REST para obtener información de videojuegos
2. **Datos Locales (Room):** Persistencia local para biblioteca personal offline
3. **Clean Architecture:** Separación clara de responsabilidades en capas
4. **Jetpack Compose:** UI moderna y declarativa
5. **Inyección de Dependencias (Koin):** Gestión de dependencias de forma escalable
